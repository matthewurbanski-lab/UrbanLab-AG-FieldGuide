How to host (GitHub Pages)

  1. Create a GitHub repo (public is simplest).
  2. Add index.html to the repo root.
  3. Add your logo file at /assets/logo.png.
  4. Repo → Settings → Pages → Source: Deploy from a branch → Branch: main → Folder: /(root).
  5. Open the URL GitHub gives you.
